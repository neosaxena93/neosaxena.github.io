========================================
   HOW TO OPEN NEO SAXENA'S RESUME
========================================

Hello! This resume is an interactive webpage saved as a file
on your computer. Here's how to open it:


--- STEP 1: Find the file ---

Look for the file named:

    neo resume.html


--- STEP 2: Open it ---

Simply DOUBLE-CLICK the file. It will open in your default
web browser (Chrome, Edge, Firefox, Safari, etc.) just like
a regular website -- no internet connection required.


--- IF DOUBLE-CLICKING DOESN'T WORK ---

Try one of these:

  Option A - Right-click the file
     1. Right-click "neo resume.html"
     2. Select "Open with"
     3. Choose your web browser (Google Chrome, Microsoft Edge, etc.)

  Option B - Drag and drop
     1. Open your web browser
     2. Drag the "neo resume.html" file into the browser window


--- TIPS ---

  * The resume has a light and dark mode toggle -- look for
    a moon/sun icon at the top of the page.

  * You can scroll through the page just like any website.

  * No software needs to be installed. If you can read this,
    you already have everything you need.


--- TROUBLE? ---

If the file looks like a wall of code instead of a styled
page, you likely opened it in a text editor by mistake.
Close it, then right-click the file and choose "Open with"
your web browser instead.
